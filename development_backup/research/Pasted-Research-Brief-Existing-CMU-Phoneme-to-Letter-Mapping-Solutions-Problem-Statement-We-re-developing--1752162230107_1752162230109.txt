Research Brief: Existing CMU Phoneme-to-Letter Mapping Solutions
Problem Statement
We're developing an educational app that needs to divide English words into syllables. We've been working with the CMU Pronouncing Dictionary but face a key challenge: the CMU dictionary breaks sounds into individual phonemes (like Y UH R for the /er/ sound), but for educational syllabification, we want to keep r-controlled vowels like "ur", "er", "ir" together as single units.
Core Challenge

CMU phonemes: Y UH R (individual sounds)
Educational need: "ur" (single r-controlled vowel unit)
Question: Has someone already solved comprehensive phoneme-to-letter mapping for educational use?

Current Solution (V2 System)
We've built a working system that:

Uses letter-based vowel cluster detection first
Validates with phoneme counts second
Handles r-controlled vowels properly: "spurious" → "spur-i-ous" ✓
Works well but may not be comprehensive

What We Need to Find
Primary Goal: Existing libraries/systems that map CMU phonemes to educational letter patterns
Specific Mapping Patterns Needed:

R-controlled vowels: ER → "er", Y UH R → "ur", IH R → "ir"
Vowel digraphs: EY → "ay", OW → "ow", AY → "ai"
Complex patterns: AO → "au", OY → "oi"
Educational vs. phonemic conflict resolution
