The **Moby Hyphenator (Open Data Set)** is a public-domain resource from the Moby Project, created by Grady Ward, and is one of the largest freely available English hyphenation lists[3][8][7].

### Key Facts

- **Content:** Contains hyphenation (syllable division) for approximately 187,175 English words and phrases[3][8][4].
- **Format:** Each entry shows a word with its syllable breaks, typically marked by a bullet character (- , character value 165 decimal, A5 hexadecimal) or, in some versions, a hyphen. For example:  
  - `at- mos- phere`  
  - `ca- pac- i- ty`  
  - `un- col- or- a- ble`[3]
- **Encoding:** Original files use MacRoman encoding, but plain text and UTF-8 versions are available[3][7].
- **No Diacritics:** Syllable breaks use regular characters, not phonetic or diacritical marks[3].
- **Licensing:** Entirely public domain—free for any use, including commercial and educational projects[3][8][4].
- **Documentation:** There is little to no documentation about the specific hyphenation decisions, but the dataset is widely used and can be customized as needed[3].

### How to Access

- **Project Gutenberg:** Download the Moby Hyphenation List for free in various formats, including plain text and HTML[7].
- **GitHub:** The full Moby Project, including the Hyphenator, is available for download and integration[8].
- **Other Mirrors:** Datahub and other open data repositories also host the dataset[4].

### How to Use

- **Integration:** You can import the file into your own database or application for offline or bulk processing.
- **Customization:** Since it is public domain and in plain text, you can edit the entries, change delimiters, or apply additional rules to suit your needs[3].
- **No API Required:** This is a static dataset, so there are no usage limits or per-request costs.

### Example Entry

| Word         | Hyphenation         |

**Summary:**  
The Moby Hyphenator is a free, public-domain, and highly flexible syllable division resource for English, ideal for educational, accessibility, and language processing projects. It can be downloaded, edited, and used without restriction[3][8][7].
