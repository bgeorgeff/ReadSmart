What is the best available dictionary that accurately divides words into syllables for pronunciation with regular letters, not schwa symbols and accents and other diacritical marks. This is for an app that breaks words into syllables to help dyslexics learn to read.


ChatGPT said:
The best available resource for dividing words into syllables using regular letters only (without IPA, schwa symbols, or diacritics)—especially for dyslexia-friendly reading instruction—is Hyphenation-based syllable dictionaries or tools that:

Reflect phonetic syllable breaks clearly,

Use only standard alphabet letters, and

Avoid IPA or complex linguistic symbols.

Here are the top options that can help you build or support an app for syllable-based word breakdown:

✅ 1. WordSmyth Dictionary
Why it’s good: Shows syllable breaks using hyphens (e.g., in-for-ma-tion) and includes pronunciation using regular letters in many entries.

URL: www.wordsmyth.net

Strengths:

Easy-to-read syllables

Especially useful for educators and reading specialists

Drawback: Limited API access unless you license it

✅ 2. HowManySyllables.com
Why it’s good: Gives syllables broken up with hyphens and includes audio and plain-text syllabification with no special characters.

URL: www.howmanysyllables.com

Strengths:

Built specifically for learners

Clean syllable division with only regular letters

Drawback: No open API access unless special permission is granted

✅ 3. CMU Pronouncing Dictionary (Simplified Version)
Why it’s good: A well-maintained open-source pronunciation dictionary from Carnegie Mellon University.

URL: http://www.speech.cs.cmu.edu/cgi-bin/cmudict

Strengths:

Accurate syllabification and stress patterns

Easily programmable (used in many TTS and decoding tools)

Tip: You can strip the phoneme codes and simply use the syllable boundaries for a phonics-focused tool.

Drawback: Uses ARPAbet phonemes, so you’ll need to convert it to letter-based output yourself.

✅ 4. NLTK’s cmudict module (for developers)
A Python tool that lets you use the CMU dictionary programmatically.

Can be modified to return only syllable breaks using standard letters, great for building educational apps.

Example transformation: "INFORMATION" → ["IN", "FOR", "MA", "TION"]

Useful for customizing syllable output with your own phonics rules.

BONUS: Make Your Own Hybrid Tool
For dyslexia apps, many developers create hybrid dictionaries by:

Using a syllable-based source (like CMU or WordSmyth)

Mapping to real spellings (not just phonemes)

Removing schwa and diacritics

Using consistent decoding rules aligned with Science of Reading principles

Final Recommendation for You
If you are building a dyslexia-focused app that teaches decoding and phonemic awareness, here’s the best combo:

Use HowManySyllables.com or WordSmyth for simple lookups or small datasets

Use CMU Pronouncing Dictionary (with a custom script to output syllables in regular letters) for scalability and accuracy

Include an audio component and visual syllable highlighting for multi-sensory learning
